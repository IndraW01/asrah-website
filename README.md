## Instalasi

1. Clone repositori ini.
2. Jalankan perintah: `npm install` atau `yarn install`.
3. Konfigurasi file .env (jika diperlukan).

## Penggunaan

1. Jalankan perintah: `npm start` atau `yarn start`.
2. Akses aplikasi di `http://localhost:3000`.
